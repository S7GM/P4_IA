# PRÁCTICA 4 INTELIGENCIA ARTIFICIAL
### Objetivo: 
#### El objetivo del programa es simular el algoritmo de Prim para construir un árbol parcial mínimo a partir de un grafo, mostrando paso a paso cómo se eligen las aristas de menor peso y generando una representación gráfica que permita visualizar de manera sencilla y clara el crecimiento del árbol mínimo durante todo el proceso.
